# Workspace Rules for creatorsfeel.com

1. Mobile-first development is mandatory.
2. Every page must be fully responsive from 320px to 1920px.
3. Mobile responsiveness is the highest priority.
4. Website speed is the highest priority.
5. SEO is the highest priority after performance.
6. Never use placeholder code or TODO comments.
7. Never redesign completed sections unless I ask.
8. Use Astro best practices.
9. Generate static pages whenever possible.
10. Use reusable components.
11. Keep JavaScript minimal.
12. Always deliver production-ready code.
13. Test every page for desktop, tablet and mobile before marking it complete.
